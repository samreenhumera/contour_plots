CONTOUR PLOTS

A contour plot is a graphical method to visualize the 3-D surface by plotting constant Z slices called "CONTOURS" in a 2-D format.

The contour plot is an alternative to a 3-D surface plot.

It uses contour lines to connect points with the same value.

The contour plot is formed by:

  • Vertical axis: Independent variable 2
  • Horizontal axis: Independent variable 1
  • Lines: iso-response values, can be calculated with the help(x,y).

A contour plot is appropriate if you want to see how some value Z changes as a function of two inputs, X and Y:

z = f(x, y)

PURPOSE OF CONTOUR PLOT

Visual Representation:

  • Contour plotting serves as a visual tool to represent complex data sets.
  • It provides a graphical representation that is easier to interpret than raw numerical data.
  • Visual patterns and trends become more apparent, aiding in understanding the dataset at a glance.

Variation in a Dataset:

  • Contour plots specifically highlight variations in datasets that involve two continuous variables.
  • These variations can include changes in magnitude, distribution, or other characteristics of the data.

Highlight Peaks, Valleys, or Areas of Interest:

  • Contour plots highlight specific features such as peaks and valleys in the dataset.
  • This makes it easier to pinpoint areas of interest, anomalies, or critical regions that may require further analysis or attention.

TYPES OF CONTOUR PLOT

Rectangular Contour Plot:

  • A rectangular contour plot, also known as a 2D contour plot or simply a contour plot, is a graphical representation of three-dimensional data on a two-dimensional plane.
  • It uses Cartesian coordinates (x, y) to represent data points, with contour lines indicating regions of constant values for a third variable.

    • In a rectangular contour plot:

  • The x-axis and y-axis represent the two independent continuous variables.
  • Contour lines, curves, or filled regions on the plot connect points with the same values of a third variable, usually represented by color or shading.

Polar Contour plot:

  • A polar contour plot is a graphical representation of three-dimensional data in a polar coordinate system.
  • Unlike rectangular coordinates (x, y), which use a horizontal and vertical axis, polar coordinates use a radius and an angle.

    • In a polar contour plot:

  • The x-axis and y-axis represent the two independent continuous variables.
  • The angle corresponds to the angular direction around the center, often measured in degrees or radians.

rad_arr = np. radians(np.linspace(0, 360, 20)) 
r_arr = np.arange(0, 1, .1)

return r * np.sin(theta)

values = func(r,theta) 

# plot the polar coordinates
fig, ax = plt.subplots(subplot_kw=dict(projection='polar')) 
ax.contourf(theta, r, values, cmap='Spectral_r')

Ternary Contour Plot:

  • A ternary contour plot is a graphical representation of three-variable data on a ternary diagram.
  • A ternary diagram is a triangular graph where the three corners represent the three components of a mixture, and points within the triangle represent compositions of these components.
  • he axes in a ternary diagram are labeled with percentages or proportions of each component.

    • In a ternary contour plot:

  • The plot is based on a triangular coordinate system, with each corner of the triangle representing a pure component, and the sides representing binary mixtures.
  • Contour lines connect points with equal values of a third variable within the ternary space. These lines indicate regions of constant concentration or some other property.
